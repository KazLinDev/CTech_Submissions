# Submission 05

## Additional Notes
- If you are trying to access the site through a mobile device, please make sure it is not on silent mode, otherwise the music won't play!
- On Desktop press 'H' to toggle the Debugger and try out some of the effects.


## Setup
You need to have NodeJS installed.
Download [Node.js](https://nodejs.org/en/download/).

Afterwards run these commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
